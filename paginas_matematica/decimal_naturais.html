Conjuntos Numéricos

Antes de iniciarmos, precisamos entender como funciona os numeros e quais tipos de numeros existem. Iremos rever esses conceitos e depois começar a utiliza-los para adição, subtração, divisão e multiplicação.

Aqui estão os tipos de números que devemos entender:

Números Naturais e Números Inteiros., porém, iremos dar enfâse apenas nos mais importantes, que serão os Números Naturais e Números Inteiros

Números Naturais

Os Números Naturais são aqueles que usamos todos os dias para contar coisas. São os números que começam do zero e vão para frente: 0, 1, 2, 3, 4, 5... e assim por diante, sem parar!

  • Se você tem uma maça, isso é o número 1.
  • Se você ganha mais uma maça, agora você tem duas maças, ou seja, o número 2.

Números inteiros

Já os Números Inteiros são como uma régua mágica que vai para os dois lados:

  • De um lado, temos os números que você ja conhece para contar: 0, 1, 2, 3, 4... que são os Números Naturais.
  • Do outro lado, existem os números negativos: -1, -2, -3, -4... Eles são como "dívidas" ou quando a gente "perde alguma coisa"
  • Dessa maneira, pode se dizer que os números inteiros são ...-4, -3, -2, -1, 0, 1, 2, 3, 4..., ou seja, é mesclado com os números naturais.
